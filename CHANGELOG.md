# CHANGELOG

## [0.2.0] - 2026-05-16

### Added
- **CSVログ保存**: 照合成功（OKかつ重複なし）の日時・1本目・2本目・結果を内部ストレージへ自動追記
- **CSVをダウンロード**: Android 共有シートでメール・Google Drive 等へ送信
- **ログをクリア**: ログファイルと重複チェックセットを同時削除
- **読み込み数設定**: 目標件数をダイアログで入力し SharedPreferences に永続化
- **進捗表示**: スタート画面に `x / N 件` を表示。完了時は `読み込み終わりました。新しい読み込み数を設定してください` を表示しスタートボタンを無効化
- **重複チェック**: 同じバーコードを2度読んだ場合、橙色で「重複」と表示し件数を増やさない。ログクリアで重複セットもリセット
- **バージョン表示**: スタート画面の下部に `v0.2.0` を表示

### Changed
- OKのみCSVに保存（NG・重複は保存しない）
- OK 結果画面のボタンを「もう一度」→「次のバーコードを読む」に変更（NG・重複は「もう一度」を維持）
- 完了時（x/N 完了）のボタンを「スタート画面へ」に変更
- ResultScreen: scannedCount・targetCount パラメータ追加、DUPLICATE（橙）対応
- ScanViewModel: logRepo・settingsRepo をコンストラクタで受け取り（テスト時は省略可能）

## [0.1.0] - 2026-05-15

### Added
- 初回リリース
- CameraX によるカメラプレビューとバーコード読み取り（ML Kit、全フォーマット対応）
- 1本目読み取り後に値を画面表示し「次へ」ボタンで確認してから2本目へ進む（CONFIRMING_FIRST フェーズ）
- OK（青）/ NG（赤）照合結果画面
- 音声フィードバック: 読み取り成功（ピッ）、一致（OK音）、不一致（NG音）
- 「次のバーコードを読む」「中止」ボタン（当時は「もう一度」）
- カメラ権限リクエストと拒否時のエラー表示
- ポートレート固定表示
