# CHANGELOG

## [Unreleased]

### Changed
- カメラ起動とOCR開始を分離。カメラ画面を開いた直後はプレビューのみ表示し、OCRは実行しない
- 「読む」ボタン押下時のみ OCR を1回実行するように変更（自動読み取りループを廃止）
- `ScanState.cameraReady` フラグと800ms自動有効化ロジックを削除
- `ScanViewModel.onCameraReady()` を削除
- 画面ガイドメッセージを「1本目：文字にカメラを合わせて「読む」を押してください」「2本目：〜」に更新

## [0.3.0] - 2026-05-16

### Added
- **OCR長チェック**: 設定した長さと異なるOCRを読んだ場合、「OCR長が違います（読んだ長さ = x）」を表示しエラーとする。0 設定時は任意の長さを受け付ける
- **ヘッダーチェック**: 設定した文字列で始まらないOCRを読んだ場合、「ヘッダーが一致しません」を表示しエラーとする。空欄時はチェックなし

### Changed
- 「読み込み数（変更）」ボタンを「設定」ボタンに統合。ダイアログで「読み込み数」「OCR長」「ヘッダー」の3項目を一括設定できるようにした
- 読み込み数が未設定（0）のときはスタートボタンを無効化し、「読み込み数を設定してください」を赤字で表示する
- SettingsRepository に ocrLength・ocrHeader を追加し SharedPreferences で永続化
- ScanViewModel に onSaveSettings(targetCount, ocrLength, ocrHeader) を追加（onSetTargetCount を廃止）

---

## [0.2.0] - 2026-05-16

### Added
- **CSVログ保存**: 照合成功（OKかつ重複なし）の日時・1本目・2本目・結果を内部ストレージへ自動追記
- **CSVをダウンロード**: Android 共有シートでメール・Google Drive 等へ送信
- **ログをクリア**: ログファイルと重複チェックセットを同時削除
- **読み込み数設定**: 目標件数をダイアログで入力し SharedPreferences に永続化
- **進捗表示**: スタート画面に `x / N 件` を表示。完了時は `読み込み終わりました。新しい読み込み数を設定してください` を表示しスタートボタンを無効化
- **重複チェック**: 同じOCRを2度読んだ場合、橙色で「重複」と表示し件数を増やさない。ログクリアで重複セットもリセット
- **バージョン表示**: スタート画面の下部に `v0.2.0` を表示

### Changed
- OKのみCSVに保存（NG・重複は保存しない）
- OK 結果画面のボタンを「もう一度」→「次のOCRを読む」に変更（NG・重複は「もう一度」を維持）
- 完了時（x/N 完了）のボタンを「スタート画面へ」に変更
- ResultScreen: scannedCount・targetCount パラメータ追加、DUPLICATE（橙）対応
- ScanViewModel: logRepo・settingsRepo をコンストラクタで受け取り（テスト時は省略可能）

## [0.1.0] - 2026-05-15

### Added
- 初回リリース
- CameraX によるカメラプレビューとOCR読み取り（ML Kit、全フォーマット対応）
- 1本目読み取り後に値を画面表示し「次へ」ボタンで確認してから2本目へ進む（CONFIRMING_FIRST フェーズ）
- OK（青）/ NG（赤）照合結果画面
- 音声フィードバック: 読み取り成功（ピッ）、一致（OK音）、不一致（NG音）
- 「次のOCRを読む」「中止」ボタン（当時は「もう一度」）
- カメラ権限リクエストと拒否時のエラー表示
- ポートレート固定表示
