# OCRリーダー Android アプリ（実証実験版）

Androidスマートフォンのカメラで文字を読み取り、認識結果を確認できるOCR実証実験用アプリです。

---

## 目的

スマートフォン内蔵カメラによるOCRが現場作業やメモ確認に耐えられるかを検証します。  
紙のラベル、印字されたメモ、帳票の一部などをその場で読み取る用途を想定した実験版です。

---

## 技術スタック

| 項目 | 採用技術 |
|------|---------|
| 言語 | Kotlin |
| UI | Jetpack Compose |
| カメラ | CameraX |
| OCR | ML Kit Text Recognition（Japanese） |
| ビルド | Gradle Kotlin DSL |
| パッケージ名 | `com.garyohosu.ocrreader` |

---

## 主な機能

- カメラ映像からOCRを実行
- 認識した文字列を2回確認して一致判定
- 文字数や先頭文字列の条件で簡易チェック
- OK / NG / 重複を色と音で通知
- OCR結果をCSVログとして保存・共有
- Telegram / Slack 連携や定期実行は、今後の拡張先として検討可能

## ビルド方法

### 前提条件

- Android Studio（最新安定版）
- JDK 17 以上
- Android SDK（API 23 以上）

cd android-app

```

```

```bash
./gradlew testDebugUnitTest
```

## 操作方法

### 基本フロー

```
アプリ起動
  ↓
スタート画面 →「設定」ボタンで読み取り回数・文字数・先頭文字列を設定
  ↓
「スタート」ボタンを押す（読み取り回数が未設定の場合は無効）
  ↓
1回目のOCR画面 → カメラに紙やラベルをかざす
  ↓
確認画面 → 認識した文字列を表示。「次へ」ボタンで進む
  ↓
2回目のOCR画面 → 同じ対象を再度読み取る
  ↓
判定画面
  ├─ 一致・未登録 → 青背景「OK」＋ OK音 → 進捗 x/N 件
  ├─ 不一致      → 赤背景「NG」＋ NG音
  └─ 一致・重複  → 橙背景「重複」＋ NG音
```

### OCR のチェック設定

| 設定 | 動作 |
|------|------|
| OCR文字数（0 = 任意） | 読み取った文字数が設定値と異なる場合はエラー |
| 先頭文字列（空 = チェックなし） | 認識結果が指定文字列で始まらない場合はエラー |

## 既知の制限事項

- OCR精度は照明・フォント・撮影距離に大きく左右されます
- 手書き文字や低解像度の文字は誤認識が増えます
- 画面は縦向き固定です
